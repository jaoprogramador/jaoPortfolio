Estructuras

Una estructura es un tipo de dato definido por el usuario, que consiste en un grupo de datos los cuales pueden ser tratados como una unidad. Las estructuras tienen algunas de las características de los arrays:

- Se puede acceder y manipular la estructura globalmente. - Los miembros de una estructura también pueden ser accedidos y manipulados individualmente. - los elementos de una estructura están almacenados en posiciones contiguas de memoria. - La inicialización de una estructura se hará de similar forma que un array.

La sintaxis a seguir para declarar una estructura es la siguiente:

struct nombre_estructura {    tipo_datos mienbro1;    tipo_datos mienbro2; }

Las variables definidas como parte de la estructura se llaman miembros. Los miembros de una estructura pueden ser cualquier variable de los siguientes tipos de datos:

- tipos de datos básicos(int, float, char...) - cadenas de caracteres - arrays - punteros - otras estructuras

Ejemplo:

struct asignaturas{
	  int analisis;
	  int programacion;
	  int sistemas;
	  }
