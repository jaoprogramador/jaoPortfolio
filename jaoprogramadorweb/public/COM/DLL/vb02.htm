Pasos para hacer una DLL

1.- Abrimos Microsoft Visual Basic y a la hora de especificar que tipo de proyecto vamos a crear seleccionamos DLL ActiveX.

2.- Ahora vendía la parte de programación como si una función o procedimiento de un modulo se tratase.

3.- Tras haber programado la dll, tendremos que generarla y guardarla, haciendo Menu->Archivo -> Generar Nombre_Proyecto_DLL.dll y guardar. Con esto habremos registrado en nuestro equipo la DLL.
