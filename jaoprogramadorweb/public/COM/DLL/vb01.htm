Introducción a DLLs

COM (Component Object Model) es la tecnología de crear componentes, sobre la que está basada Windows y puede ser utilizado independientemente del lenguaje.

Así pues, un componente es un objeto que nos muestra que servicios implementa pero no como. Es decir, una control Textbox de Visual Basic tiene propiedades como Text, Caption ... que nosotros utilizamos para programar pero que internamente e invisibles para nosotros, tienen a su vez una programación.

Un ejemplo de COM son las DLLs.

Las ventajas de son la reutilización de código de una forma elegante sin tener que copiar-pegar y reajustar, el consumo de memoria se minimiza entre DLLs usadas ppor distintos programas y poder crear componentes desde distintos lenguajes de programación.

Las desventajas serían la complejidad de desarrollo de los propios componentes.
