Pasos para hacer una OCX

1.- Abrimos Microsoft Visual Basic y a la hora de especificar que tipo de proyecto vamos a crear seleccionamos Control ActiveX.

2.- Ahora vendía la parte de programación y diseño del control como si una función o procedmiento de un modulo se tratase.

3.- Tras haber programado el control ActiveX, tendremos que generar el control OCX y guardarlo, haciendo Menu->Archivo -> Generar Nombre_Proyecto_OCX.ocx y guardar. Con esto habremos registrado en nuestro equipo la el nuevo componente ActiveX.
