Cadenas de caracteres

1.- Ejemplos de utilizar tres funciones LEN, SUBSTR y PATINDEX de cadena para extraer los apellidos, elminando el nombre de los autores.

use libreria
select titulo,substring(autor,patindex('%%',autor)+1,len(autor))as 'Apellidos del autor' from libros

  

Numéricos

2 .- Ejemplos de como con la función MAX y MIN mostrar cual es el precio del libro mas caro y mas barato. A continuación almacena el precio máximo en una variable y utilizala para obtener el título del libro mas caro.

use libreria
select count(titulo)as 'nº',max(precio) as 'Máximo',min(precio)as 'Mínimo' from libros
declare @precioMaximo money
select @precioMaximo =max(precio)from libros
select titulo,precio from libros where precio=@precioMaximo
