IMD

1.- Base de datos IMD que contiene dos tablas: ABONOS y USUARIOS, para almacenar los registros que pertenecen a un club y el tipo de abono que tienen para acceder a ella.

  

CENTRO

3 .- Base de datos CENTRO que contiene dos tablas: ALUMNOS y MATRICULAS, para almacenar los registros de de los distintos alumnos que estudian en un centro y el tipo de matrícula que tienen.

  

LIBRERIA

3 .- Base de datos LIBRERIA que contiene cinco tablas, para almacenar los registros de de los distintos libros, editorial y distintos distribuidores.
