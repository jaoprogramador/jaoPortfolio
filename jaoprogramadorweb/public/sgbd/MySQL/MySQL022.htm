Valores literales
Este capítulo trata sobre las reglas a seguir cuando se escriban los siguientes elementos de sentencias SQL durante el uso de
MySQL:
• Valores literales, como cadenas y números.
• Identificadores, como nombres de tablas y columans.
• Variables de usuario y de sistema
• Comentarios
• Palabras reservadas

Valores literales


En esta sección se trata la escritura de valores literales en MySQL. Estos incluyen a las cadenas, números, valores hexadeciales, valores
booleanos y NULL. También se ocupa de las distintas situaciones (algunas muy propensas a error) en las que se puede incurrir
al manejar estos tipos básicos de MySQL.
