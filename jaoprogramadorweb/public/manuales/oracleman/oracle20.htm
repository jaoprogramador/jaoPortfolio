Programación Oracle con PL/SQL

El lenguaje de programación Oracle, llamado PL/SQL, es un lenguaje portable, procedural y de transacción muy potente y de fácil manejo, con las siguientes características fundamentales:

1.- Incluye todos los comandos de SQL estudiados:

- SELECT - INSERT - UPDATE - DELETE

2.- Es una extensión de SQL, ya que este es un lenguaje no completo dado que no incluye las herramientas clásicas de programación. Por eso, PL/SQL amplia las posibilidades al incorporar las siguientes sentencias:

- Control condicional IF ... THEN ... ELSE ... ENDIF - Ciclos FOR ... LOOP WHILE ... LOOP

3.- Incorpora opciones avanzadas en: - Control y tratamiento de errores llamado excepciones. - Manejo de cursores. - Variedad de procedimientos y funciones empaquetadas incorporadas en el módulo SQL*Forms para la programación de disparadores (Trigger) y procedimientos del usuario (Procedure).
