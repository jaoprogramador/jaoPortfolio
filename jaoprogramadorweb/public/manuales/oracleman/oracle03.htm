Ingreso de datos

Una vez creadas las tablas, índices y secuencias, estamos en condiciones de ingresar datos en la tabla. El ingreso, modificación y eliminación de registros se realiza fundamentalmente con el diseño de pantallas (formularios)desde el modulo SQLFORMS. No obstante, en éste solo veremos el uso de los mandatos INSERT, UPDATE Y DELETE. Para realizar un ingreso en la tabla deberemos insertar:

Nombre de la tabla (acciones) Columnas 
INSERT INTO CLIENTES 
VALUES(
   codigo_clientes.NEXTVAL),
   'PINTURERIAS PROPIOS',
   '67876098',
   TO_DATE('10/03/03','DD/MM/YY'),
   'Gipuzkoa nº 30',
   'Librería'
);

Anotación

Como se podrá observar, en este ejemplo no se especificó la lista de columnas a insertar, lo que indica que se van a insertar datos en todas las columnas. Por lo tanto, los valores para cada columna se tienen que ingresar en el orden en que están definidos en la tabla. Además es de destacar que la palabra reservada VALUES indica la lista de valores a ingresar; que los datos tipo carácter van entre comillas; que la fecha se registra como una cadena de caracteres usando la función TO_DATE (encargada de transforma la cadena de caracteres 10/03/03 en fecha, a partir de un formato de fecha específico: DD/MM/YY). También observamos que a la columna código se le asigno el siguiente valor de la secuencia codigo_cliente.NEXTVAL. En el caso de que solo se asignarán valores a algunas columnas se debe dar la lista de columnas como se muestra a continuación:

INSERT INTO CLIENTES (codigo, nombre, telefono, fecha) VALUES
(codigo_cliente.NEXTVAL, 'Casa Julen Gerrero', '8888888',
TO_DATE('15/04/05','DD/MM/YY'));
