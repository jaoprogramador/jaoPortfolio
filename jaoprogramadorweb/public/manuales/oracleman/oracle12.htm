Consultas SQL en Oracle

Ahora pasaremos a estudiar la consulta y selección de registros con el lenguaje estándar para bases de datos relacionales SQL (Structured Query Languague = Lenguaje de Consulta Estructurado).

La ventaja principal del SQL, es su capacidad de combinar sencillez y facilidad con potencia y eficiencia, conteniendo un conjunto de herramientas que optimizan las consultas.

Vale la pena destacar que, los conceptos a estudiar son específicos de ORACLE, también son útiles para cualquier programador que esté trabajando con algún sowtware que contenga SQL.
