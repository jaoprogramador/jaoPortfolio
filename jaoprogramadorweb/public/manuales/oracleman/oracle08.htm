Listar estructuras de las tablas

Para obtener la estructura o descripción de una tabla el mandato que se debe emplear es: DESC o DESCRIBE.

SQL> DESC clientes;

Con lo que obtendremos el resultado:

Name Null? Type
numero not null number(38)
fecha not null date
nombre not null char(30)
telefono not null char(20)
dirección   char(10)
anotación   long
