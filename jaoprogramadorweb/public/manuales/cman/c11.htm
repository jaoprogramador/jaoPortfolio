Punteros

Un puntero es la dirección de un objeto en memoria. Para obtener la dirección de algo, se utiliza el operador &. Para obtener aquello a lo que apunta una dirección ,se utiliza el operador *. La sintaxis para declarar una variable puntero es la siguiente: Tipo de aquello a lo que se apunta *nombre_puntero. Ejemplo: 

char *p,c;
	  c='a';
	  p&cc;
