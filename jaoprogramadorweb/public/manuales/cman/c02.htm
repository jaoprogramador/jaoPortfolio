Datos fundamentales

Se pueden dividir en tres apartados:

1.- Datos simples: que a su vez pueden ser:
- Caracteres - Números enteros - Números reales
2.- Datos derivados: (obtenidos a partir de datos fundamentales)que a su vez pueden ser:
- Punteros - Arrays - Funciones - Estructuras
3.- Construcciones fundamentales:
- Agrupación de sentencias - Toma de decisiones(if-else) - Selección de un caso entre varios (switch) - Bucles (While, For, Do) - Salida anticipada de un bloque
