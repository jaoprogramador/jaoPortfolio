Entorno de Visual Basic

Cuando se ejecuta Visual Basic se pueden ver varias ventanas.

- Ventana Principal. Es la que contiene el conjunto de ventanas: ventana de diseño, el cuadro de herramientas y el explorador de proyectos, la ventana de propiedades y la ventana de posición del formulario.
- Ventana de diseño y editor de código. La ventana de diseño es la que nos permite ver y diseñar mediante el ratón el aspecto que tendrá el programa, mientras que editor de código es donde escribiremos el código que contendrá internamente la aplicación.
- Cuadro de Herramientas. El cuadro de herramientas contiene los controles que podemos añadir directamente al formulario de la aplicación.
- Explorador de proyectos. Esta ventana nos permite cambiar de vista de diseño a la vista de código, así como ir seleccionando los distintos ficheros que contiene cada proyecto o aplicación.
- Ventana de propiedades. Esta ventana nos muestra las propiedades del control que está seleccionado en modo diseño.
