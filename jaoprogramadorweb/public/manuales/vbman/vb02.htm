Controles de una aplicación

Una aplicación está formada principalmente por formularios, controles, eventos, métodos e instrucciones ejecutables.

- Formularios es la ventana que vemos cuando abrimos cualquier aplicación de windows y en ella situamos el resto de controles.
- Controles. Son los objetos que windows u otras aplicaciones tienen para facilitar la interacción con la aplicación. Estos pueden ser botones, cuadros de texto, etiquetas, listas desplegables...
- Eventos. Es el código que se ejecuta cuando realizamos una acción, como al hacer click sobre un botón.
- Métodos. Son acciones asociadas al control, pero ya programadas, cuyo código no podemos modificar.
