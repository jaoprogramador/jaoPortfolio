Introducción

El siguiente elemento es uno de los mas importantes del lenguaje HTML, ya que el que realmente permite "navegar" por uno o varios documentos, que pueden residir en cualquier parte, dando sin embargo, la sensación de que se trata de un solo documento.

Esto se consigue definiendo enlaces hipertextuales del Web mediante el elemento <A> y sus atributos: NAME, HREF Y TARGET. El lector podrá por tanto explorar el documento picando con el ratón sobre las zonas activas definidas, denominadas hipertexto. Estas zonas activas pueden corresponder a un grupo de caracteres, una imagen o bien una porción de ella. En cualquier caso el principio es el mismo: asociar a la zona activa la dirección URL del documento que sustituirá al visualizado cuando se pulse con el ratón sobre esa zona.

Un puntero puede colocarse en cualquier lugar del texto. Puede ser un elemento de una vista, o texto normal; puede estar enriquecido con atributos de estilo físico, lógico o de párrafo. Sin embargo no es necesario darle un atributo para hacerle reconocible como tal; los enlaces activables se destacan automáticamente (color y subrayado), siempre y cuando el usuario no relalice una parametrización especial del browser.

La definición por defecto del color de os enlaces de un documento puede ser modificada por el elemento <BODY> y los atributos LINK, ALINK y VLINK.Para especificar la partida y la llegada de un enlace hipertexto se define:

- Ancla de partida: es la zona activa sobre la que el lector pulsará con el ratón para llamar a una nueva página. Este ancla se define mediante el atributo HREF. - Ancla de llegada: es una zona inactiva que especifica el punto de llegada de un enlace hipertexto. Esta es pues una dirección. Este ancla se define mediante el atributo NAME.

A

El elemento <A> se utiliza para definir losa enlaces hipertexto de los documento HTML. Este elemento no se utiliza solo, por tanto siempre vendrá acompañado por, al menos, uno de sus dos atributos principales:

- El atributo HREF que define un ancla de partida. - El atributo NAME que define un ancla de llegada. - El atributo TARGET para ordenar la apertura de una nueva ventana del browser con la página indicada por HREF.

A continuación se detallan cada uno de los atributos:

El atributo NAME

Este atributo define un punto de llegada en el documento destino asignándole un nombre o etiqueta: <A NAME ="etiqueta">Zona no activable</A> Luego mediante el atributo HREF desde el fichero de partida se indicará el documento y la posición dentro de dicho documento a la quiere acceder: <A HREF="Documento#etiqueta"> Zona activable</A> Hay que tener en cuenta que que la etiqueta debe estar compuesta por un texto sin espacios en blanco, caracteres especiales, ni caracteres codificados. Dicha etiqueta no deberá estar repetida dentro del mismo documento destino.

El atributo HREF

Este atributo define un ancla de partida hacia un enlace externo o interno, es decir, crea un enlace hacia un nuevo documento propuesto por dicho servidor o hacia un punto determinado en el texto actual. La sintáxis típica es: <A HREF="http:///www.google.com"> Buscador típico de internet </A> Buscador típico de internet

El atributo TARGET

Este atributo ordena la apertura de una nueva ventana con el documento indicado por el atributo HREF. Ejemplo: <a href="htm07.htm"target="mainFrame" >Siguiente</a> Siguiente Si pulsas con el ratón sobre el enlaza se visualizara una página de "TABLAS" en el mismo navegador.

Enlaces a puntos internos a un documento

Para acceder a un punto determinado de un documento se debe definir el ancla de llegada en el documento destino mediante el atributo NAME, como se vio anteriormente. Luego se debe indicar la dirección del documento destino y del punto de comienzo. En el documento de partida, mediante el atributo HREF. La sintáxis es: En el punto adecuado del documento destino: <A NAME="etiqueta"> </A>

En el documento de partida: <A HREF="#etiqueta"> Zona activable </A>

Todo esto es muy útil cuando se trabaja con documentos grandes llenos de secciones. Ejemplos: <A HREF="#Inicio">Inicio del documento</A> Inicio del documento

Pulsando sobre el enlace "Inicio del documento nos situará en el comienzo del documento actual."
