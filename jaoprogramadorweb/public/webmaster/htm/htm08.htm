Introducción

Los Frames permiten dividir la pantalla en distintas áreas, unas independientes de otras. Cada una de estas zonas son capaces de mostrar de mostrar código HTML, como en las celdas de una tabla. En el caso de los frames cada zona de la pantalla es un documento HTML propio. La funcionalidad de los Frames es dedicar cada uno de ellos a cierta tareas bien definidas, como por ejemplo tener en una zona un menú que casi nunca será necesario recargar. La peculiaridad de los marcos, a nivel de definición es que a diferencia de una página normal, cuyo cuerpo va definido por las marcas <BODY> y </BODY>, es que los marcos van definidos por las marcas <FRAMESET> y </FRAMESET>. El elemento Frameset permitirá precisar la forma de los diferentes paneles por medio de dos atributos: ROWS y COLS.

FRAMESET

Este elemento sustituye al elemento <BODY> y permite dividir una zona en subzonas, bien verticalmente, bien horizontalmente. Esta marca posee dos atributos:

- ROWS: Se utiliza para dividir la zona en subzonas horizontales. La sintaxis es: ROWS="altura_zona1,altura_zona2..., altura_zonaN" - COLS: Se utiliza para dividir la zona en subzonas verticales. La sintaxis es: COLS="altura_zona1,altura_zona2..., altura_zonaN"

 

FRAME

Es la marca utilizada para caracterizar las zonas definidas mediante la marca <FRAMESET>. Esta caracterización se realiza mediante los siguientes atributos:

- SCR: Indica el URL del documento que debe montarse en esa zona. Si el atributo no se indica, entonces la zona estará vacía. - NAME: Se utiliza para nombrar la zona a fin de que pueda convertirse en el destino de cualquier enlace de hipertexto. La sintaxis del atributo es: NAME="nombre_de_la_zona" - MARGINWIDTH: Se utiliza para precisar el número de píxels entre los bordes izquierdo y derecho de la zona y el documento HTML a visualizar, en definitiva marca el margen izquierdo. La sintaxis es: MARGINWIDTH="n" - MARGINHEIGHT: Se utiliza para el número de pixels entre los bodis superior e inferior. de la zona y el documento HTML a visualizar. La sintaxis es: MARGINHEIGHT="n" - SCROLLING: Indica si la zona debe de poseer una barra de desplazamiento. Su sintaxis es: SCROLLING="yes/no/auto" - NORESIZE: Indica que el usuario no puede modificar el tamaño de la zona.

NOFRAMES

Esta marca indica al browser que incapaz de gestionar los frames el texto que debe presentar al usuario en lugar de los paneles. Dicho texto irá entre las marcas <NOFRAMES> Y </NOFRAMES>.

EL ATRIBUTO TARGET

Este nuevo atributo afecta a las marcas que caracterizan los enlaces de hipertexto, en particular a las marcas <A y <FORM>. TARGET permite precisar el el nombre de la zona que debe recibir el el documento correspondiente al enlace. Por ejemplo: <a href="/dossier.html" target="zonadossier">ver el dossier</a> El atributo TARGET existe para indicar al browser que, en respuesta a un click, debe mostrar el documento dossier.html en el panel cuyo nombre es "zonadossier". El atributo TARGET puede tomar los siguientes valores:

_blank, indica que se cargará una nueva ventana con el fin de mostrar el documento. _selft, indica que el nuevo documento se cargará donde se encuentra el enlace hipertexto. _top, indica al browser que suprima todos los paneles existentes y visualice el documento de forma normal. _parent,indica al browser que visualice el documento ocupando toda la superficie de la zona en la que se ha visualizado el documento que contiene el enlace.
