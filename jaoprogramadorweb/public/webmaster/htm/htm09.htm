Introducción

Los formularios generan en la pantalla cuadros de diálogo con el lector permitiendo así la interacción con el usuario para consultar las bases de datos, solicitudes de documentación.... Se podrán tener zonas en las que se introducirá un texto, casillas de verificaciónm, listas de selección, etc... Para la creación de formularios se utilizan las siguientes marcas:

- FORM que delimita el comienzo y el fin de la definición del formulario. - INPUT que permite crear diferentes tipos de entradas: campo de edición y diversos tipos de botones. - SELECT que permite crear listas, menús desplegables y listas con barras de desplazamiento. - TEXTAREA que genera una zona de edición de texto libre.

Atributos comunes

El atributo NAME define el nombre que permitirá al script identificar el origen de datos. El atributo Value puede ser definido para un campo de:

- Texto - Botón SUBMIT - Botón RADIO y CHECKBOX

FORM

La marca <FORM> y </FORM> definen un formulario, y entre ellas se situarán todas las marcas que generen los diversos elementos que componen un formulario. Esta marca debe ir acompañada por dos atributos:

- METHOD, al que puede darse el valor post o get. - ACTION define la acción a ejecutar cuando se pulse el botón de sumisión.

INPUT

La marca <INPUT> servirá para definir campos para entrar un texto y botones que permiten escoger opciones. Permite varios atributos:

- SIZE, define la longitud de la ventana. - MAXLENGTH define la máxima longitud de la cadena

El atributo TYPE asociado a la marca INPUT permite la selección del elemento de entrada, y puede tomar los valores:
- TEXT define una zona de entrada de texto simple. - SUBMIT desencadena el envío del formulario hacia el script. - RESET permite borrar los datos ya entrados. - PASSWORD permite crear una palabra clave de forma confidencial. - CHEKBOX crea un bloque de botones que permite una selección múltiple de opciones. - RADIO Crea un bloque de botones que permiten una selección exclusiva entre varias ordenes. - HIDDEN sirve para pasar datos de un formulario a otro sin que se vea nada en pantalla - IMAGE hace que el visualizador presente una imagen que es sensible al ratón

SELECT

La marca <SELECT> permite generar listas de selección simple o de selección variable. Se programa con una lista en la que los items se especifican con la marca <OPTION>.

TEXTAREA

La marca <TEXTAREA> permite crear una ventana con barras de desplazamiento horizontales y verticales en las que se podrá escribir texto. El tamaño dado a los atributos ROWS y COLS delimita el tamaño de esta ventana.

Es posible definir un texto (entre las marcas <TEXTAREA></TEXTAREA>) que el usuario podrá reemplazar o completar con su propio texto.
