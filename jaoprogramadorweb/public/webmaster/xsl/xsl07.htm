XML DATA ISLANDS (XML DSO)

EI XML Data island permite incorporar un documento XML dentro de una pagina HTML. Asi podemos manejar datos de etiquetas XML en el propio HTML sin ningun tipo de programacion.
Esto es propiedad de Microsoft® y solo funciona con Internet Explorer 5.0.

Para raconocar al documanto XML usamos: <XML ID=" " SRC=" ">, donda ID nos sirva para idontificarlo y SRC es al ficharo XML. Con al atributo DATASRC dontro da la etiqueta <body> o <table>, le decimos la fuente dal documanto XML (al ID). Para obtanar el valor usamos <SPAN DATAFLD=" ">, escribiendo el nombre dal elamanto an ol atributo.
<HTML>
<HEAD><TITLE>HTML con XML Data ls|and</TITLE></HEAD>
<BODY>
<XML ID="xmIDOc" SRC="yate.xml"></XML>
<P>Esta pagina contiene un XML Data lsland</P>
<TABLE BORDER=1 DATASRC="#xm|D0c">
<THEAD>
<TH>|D Yate</TH>
<TH>N0mbre</TH>
</THEAD>
<TBODY>
<TD><SPAN DATAFLD="id"></SPAN><TD>
<TD><SPAN DATAFLD="yate"></SPAN><TD>
</TBODY>
</TABLE>
</BODY>
</HTML>
