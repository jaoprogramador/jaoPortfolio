Sintaxis básica

La sintaxis basica es el selector (que puede ser una etiqueta , una case [cass]o un identificador[id])
seguido de propiedades o lista de propiedades y el valor asignado:
selector{[propiedad:valor];...}
Por ejemplo, si queremos definir una justificación total para los párrafod de la página tendremos que escribir el siguiente código:
<Style type="text/css">
p{text-align:justify}
</Style>
La asignación de un valor a un determinado estilo se hace utilizano los dos puntos y no el signo de igualdad.
Es de destacar que en esta sintaxis no se diferenciae entre mayusculas y minusculas. Si se hacen asignaciones de varias caracteristicas
del texto, estas se separan por punto y coma.
Si se desea declarar el mismo estilo para varias etiquetas de tal forma que el estilo definido con lo svalores para los diferentes atributos afecte a todas las etiquetas, se usará:
<Style>
etiqueta, etiqueta2,... etiquetaN{estilo:valor}
</Style>
Por ejemplo:
p, tr, td , pre{color:green}
