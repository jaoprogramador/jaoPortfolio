Introducción a hojas CSS

Para proporcionar un diseño a la página es conveniente utilizar el lenguaje de hojas de estilo
en cascada o CSS(Cascading Style Sheet). Mediante este lenguaje podremos modificar el estilo que
se le aplica a los distintos contenidos de nuestras paginas web sin tener que estar continuamente reescribiendo código.
Tenemos que tener en cuenta que en el arbol de etiquetas de un documento se produce herencia de los estilos, es decir
se pueden especificar estilos globales en la etiqueta body , y así en las etiquetas de los nibleles inferiores sólo será
necesario sobreescribir los estilos que se heredan.

Ejemplo:
- Para hacer referenciar una hoja de estilos desde un documento xml:
<?xml-stylesheet type="text/css" href="nombre.css"?>
