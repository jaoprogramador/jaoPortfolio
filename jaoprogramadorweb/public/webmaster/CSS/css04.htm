Selectores CSS

CSSS funciona a base de reglas , es decir, declaraciones sobre el estilo de uno o mas elementos. Las hojas de estilo están compuestas por una o mas de esas reglas
aplicadas a un documento HTML o XML. La regla tiene dos partes: un selector y la declaración. A su vez la declaración está compuesta por una propiedad y el valor que se le asigne.
h1{color:red};
h1 es el selector, mientras que {color:red} es la declaracion
El selector funciona como enlace entre el documento y el estilo, especificando los elementos que se van a ver afectados por esa declaración . La declaración es la parte de la regla
que establecerá cual será el efecto. En el ejemplo anterior, el selector h1 indica que todos elementos h1 se verán afectados por la declaración donde se establece que la propiedad
color va a tener el valor red(rojo) para todos los elementos h1 del documenton que este vinculado a esa hoha de estilos.

Como hemos dicho hay tres clases de selectores:
- etiquetas:
<style type="text/css">
<!--
a{text-decoration:none;}
>
</style>
- class: Dado que se puede necesitar el uso de diferentes versiones de estilos de un mismo elemento, CSS permite la especificación de clases. Para definer una clase se añade un nombre
de clase al nombre de la etiqueta en cuestión
mediante la notación punto.

Definición:
<style type="text/css">
<!--
.clsRojo{color:red}
h1.azul{font-size:24pt;color:blue}
h1.verde{font-size:10pt;color:green}
-->
</style>
Utilización:
<body>
<b class="clsRojo">Juantxu Arana Olalde</b>
<i class="clsRojo">=====================</i>
<h1 class="azul">JAO en Azul</h1>
<h1 class="verde">JAO en Verde</h1>
</body>

- ID: Los identificadores funciona ee una forma similar a las clases, pero están limitados a la utilización para un solo elemento. Se utiliza # en lugar del punto.

Definición:
<style type="text/css">
<!--
#idBoldItal{font-weight:bold;font-style:italic}
#titulo{text-align:center;color:blue}
-->
</style>
Utilización:
<body>
<b id="idBoldItal">J A O</b>
<H2 id="titulo">J u a n t x u </H">
</body>
