Funciones para elementos

Prototype define funciones muy útiles para manipular los elementos incluidos en las páginas HTML. Cualquier elemento obtenido mediante la función $() puede hacer uso de las siguientes funciones:
Element.visible(): devuelve true/false si el elemento es visible/oculto (devuelve true para los campos tipo hidden)
Element.show() y Element.hide(): muestra y oculta el elemento indicado
Element.toggle(): si el elemento es visible, lo oculta. Si es elemento está oculto, lo muestra Element.scrollTo(): baja o sube el scroll de la página hasta la posición del elemento indicado Element.getStyle() y Element.setStyle(): obtiene/establece el valor del estilo CSS del elemento (el estilo completo, no la propiedad className)
Element.classNames(), Element.hasClassName(), Element.addClassName(),
Element.removeClassName(): obtiene los class del elemento, devuelve true/false si incluye un determinado class, añade un class al elemento y elimina el class al elemento respectivamente Todas las funciones anteriores se pueden invocar de dos formas diferentes:
// Las dos instrucciones son equivalentes
Element.toggle('principal');
$('principal').toggle()
