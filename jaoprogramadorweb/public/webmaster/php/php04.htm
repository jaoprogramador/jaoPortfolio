Introducción

El indicativo de variable es siempre el signo de dólar $

php admite variables numéricas, de cadena y arrays. Para crearlas basta teclear el nombre de la variable seguida de su valor: $num = 12; $cad = “texto”;

Las variables numéricas pueden ser enteras; de punto flotante o dobles, de punto flotante o no. El tipo se va a adaptar al valor que introduzcamos. $num = 6; --> Crea una variable entera $deci = 1.0; --> Crea una variable de punto flotante

También podemos declarar las variables indicando su tipo pero es más una pérdida de tiempo que otra cosa $numero = (integer) 5; $cadena = (string) “Soy una cadena”;

Incremento y decremento

Veamos un ejemplo con variables: <?php $num=17; echo $num," Muestra valor de ", "$","num", "\n"; echo $num++," Muestra el nº e incrementa en 1\n"; echo $num," Muestra nº incrementado\n"; echo ++$num, " Incrementa en 1 y muestra el numero\n"; print ( " print también sirve para mostrar cosas\n"); ?> También existen operadores de decremento ($var--) y predecremento (--$var) que funcionan del mismo modo que los de incremento y preincremento.

Variables numéricas

Como ejemplo vamos a crear un srcip que calcule el coste de una compra de x unidades a la que le aplican un descuento del n%. Este script nos devolverá tanto el coste total como el importe de cada mensualidad si decidimos pagarlo en 12 meses.

<HTML> <HEAD> <TITLE>Una de números</TITLE> </HEAD> <BODY> <?php

/* $Cantidad se debe pasar a esta página mediante un formulario o en la URL $Descuento es opcional. */ $Coste = 2000.00; $Impuesto = 0.06 //El impuesto será un 6% $TotalCoste = $Coste * $Cantidad; $Impuesto = $Impuesto + 1; // $Impuesto será ahora 1.06. $TotalCoste = $TotalCoste - $Descuento; $TotalCoste = $TotalCoste * $Impuesto; $Plazos = $TotalCoste / 12; // Imprime los resultados print ("Has comprado un total de $Cantidad unidades at \$$Coste cada uno.\n<P>"); print ("El total con impuestos menos \$$Descuento de descuento, supone \$$TotalCoste.\n<P>"); print ("Puedes hacer el pago en 12 mensualidades de \$$Plazos cada una.\n<P>"); ?> </BODY> </HTML>

Para comprobar el script añadiremos al URL los valores para las variables Coste y Descuento.

El aspecto no es muy bueno. Para mejorarlo podemos utilizar la función printf() que nos formateará la salida a nuestro gusto:

<HTML> <HEAD> <TITLE>Otra de números</TITLE> </HEAD> <BODY> <?php /* $Cantidad se debe pasar a esta página mediante un formulario o en la URL $Descuento es opcional. */ $Coste = 2000.00; $Impuesto = 0.06; //El impuesto será un 6% $TotalCoste = $Coste * $Cantidad; $Impuesto = $Impuesto + 1; // $Impuesto será ahora 1.06. $TotalCoste = $TotalCoste - $Descuento; $TotalCoste = $TotalCoste * $Impuesto; $Plazos = $TotalCoste / 12; // Imprime los resultados. print ("Has comprado un total de $Cantidad unidades at \$$Coste cada uno.\n<P>"); print ("El total con impuestos menos \$$Descuento de descuento, supone $"); printf ("%01.2f",$TotalCoste); print (". \n<p>Puedes hacer el pago en 12 mensualidades de $"); printf ("%01.2f",$Plazos); print (" cada una.\n<P>"); ?> </BODY> </HTML>

La línea printf ("%01.2f",$TotalCoste); formatea la salida de manera que muestre un número de punto flotante, con al menos un dígito para la parte entera y dos para la decimal; el 0 inicial indica que rellene con ceros los espacios extra.

También podríamos haber utilizado sprintf() que no envía los resultados al navegador sino que cambia la variable sin imprimirla. Para ello deberíamos sustituir las líneas: print (". \n<p>Puedes hacer el pago en 12 mensualidades de $"); printf ("%01.2f",$Plazos); por: $Plazos = sprintf ("%01.2f", $Plazos); print (". \n<p>Puedes hacer el pago en 12 mensualidades de \$$Plazos cada una.\n<P>");

Para poder operar directamente con el valor del impuesto utilizamos $Impuesto = $Impuesto + 1; Otra forma de hacer lo mismo es $Impuesto++ con lo cual incrementaremos en un entero su valor. Como ya vimos anteriormente, si quisieramos disminuir su valor utilizaríamos $Impuesto--

En cuanto a la prioridad de las operaciones aritméticas decir que lo más recomendable es utilizar paréntesis para evitar resultados inesperados. Las líneas: $TotalCoste = $TotalCoste - $Descuento; $TotalCoste = $TotalCoste * $Impuesto; pueden sustituirse por: $TotalCoste = ($TotalCoste - $Descuento) * $Impuesto; . De esta forma garantizamos la prioridad de las operaciones.

Funciones matemáticas

Otras funciones matemáticas son round () --> Redondea al entero más próximo --> $n = round (22.34) // $n vale 22 --> $n = round (22.34, 1) // $n vale 22.3 ceil () --> Redondea al al entero superior floor () --> Redondea al entero inferior

abs () --> Devuelve el valor absoluto --> $n = abs (-33) // $n vale 33 rand () --> Devuelve un nº al azar Para poder utilizarla es necesario inicializarla previamente. Para hacerlo se utiliza srand(). Según el manual de php la forma de conseguir números lo más aleatorios posible se debe utilizar el siguiente código: srand ((double) microtime () * 1000000); $Naleatorio = rand();

Si queremos obtener un nº limitado a un intervalo determinado, por ejemplo de 0 a 10: $Naleatorio = rand(0, 10);
