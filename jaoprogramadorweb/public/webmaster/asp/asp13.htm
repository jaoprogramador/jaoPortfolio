Conectar con BDs en ASP

Conectar el ODBC

Cuando nos queremos conectar a una base de datos del servidor, tenemos la posibilidad de hacerlo por medio de una DNS o no. Si queremos hacerlo con un ODBC tras haber seleccionado la base de datos desde Herramientas Administrativas y haberle dado un nombre basta con insertar en el las etiquetas del BODY las siguientes líneas:

set con=server.CreateObject("ADODB.CONNECTION")
con.open("coches")'nombre de la DNS a la BD

deberán encontrarse entre <%...%> y para liberar memoria utilizaremos:

set con=nothing
set rec=nothing
