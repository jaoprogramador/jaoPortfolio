Introducción a ASP

Introducción

Asp es un lenguaje basado en visual basic script (VBS), que necesita de un servidor , Internet Information Server.

Asp es un lenguaje dinámico que genera páginas html a partir de la información de una base de datos y los resultados se presentan en servidores web (IIS) a los que tienen acceso los usuarios.

El objetido de este lenguaje, a simples rasgos, es acceder mediante un formulario a la base de datos , a través de un objeto ADO, y extraer información presentandola en formato html.

La generación de las tablas dinámicas se hará con VBS, el código estático con HTML. El contenido se genera en el servidor (html+VBS+…), el trabajo se hace en el servidor, y se envia HTML(mas lo que se quiera) al cliente. El resultado de la página Asp es el que ve el cliente en el navegador..
