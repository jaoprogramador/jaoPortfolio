Objeto Recordset en ASP

Propiedades y métodos del objeto Recordset

Se utiliza para almacenar datos de la base de datos y después manipularlos. Para crearlo, se utiliza de la siguiente manera:

set rec= server.createobject(“ADODB.recordset”)

Propiedades:

• Rec.AbsolutePosition= posicion a la que me quiero mover en BBDD (empieza en 1)
• Rec.PageSize= numero de registros por pagina (por defecto 10)
• Rec.AbsolutePage= posiciona en la pagina en la que se quiere empezar
• Rec.BOF inicio de fichero
• Rec.EOF final de fichero
• Rec.MaxRecords= numero de registros maximo de la base de datos
• Rec.PageCount devuelve cuantas paginas tiene la base de datos
• Rec.RecordCount devuelve el numero de registros de la base de datos

Métodos

• Rec=con.execute(“instrucciones SQL”)
Si la sql es una select devolvera una serie de registros que habra que almacenar en una variable recordset.
Si la sql no devuelve registros, es decir , insert , delete, devolvera -1 y no es necesario guardar en una variable.
• Rec.MoveNext pasa el cursor al siguiente registro
• Rec.MovePrevious pasa el cursor al registro anterior
• Rec.MoveFirst pasa el cursor al primer registro
• Rec.MoveLast pasa el cursor al ultimo registro
