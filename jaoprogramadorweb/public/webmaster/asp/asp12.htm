Formularios en ASP

Formularios y sus atributos

Html tiene la etiqueta <form> </form> para hacer formularios.

Method = get / / post (metodos de envio de datos del formulario). Por defecto se usa el post
Action = nombre del fichero (por ejemplo “datos.asp”) al que quiero enviar los datos para que los trate.

En asp tomaremos unos datos en un formulario asp y lo enviaremos a otra pagina asp, que los recoge y opera con ellas o los envia a la base de datos.

Ejemplos

Ahora veremos un ejemplo de como enviar datos de un formulario a otra pagina y mostrarlos:

<%@LANGUAGE="VBSCRIPT" CODEPAGE="1252"%>
<html>
<head>
<title>Documento sin t&iacute;tulo</title>
<meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1">
</head>

<body>
<form method=post action="datos_Ej16.asp">
Numero1:<input type="text" size=20 NAME="n1"><br>
Numero2:<input type="text" size=20 NAME="n2"><br>
<input type="submit" name="enviar" value="enviar">
</form>
<%
%>
</body>
</html>

En la página "datos_Ej16.asp" que recibirá los valores, para ver los valores tan solo hace falta poner dentro de las etiquetas del Body:

<%
response.write(cint(request.form("n1"))+cint(request.form("n2")))
%>

Si quisieramos validar los datos de un formulario , como el del caso anterior, nos bastaría con poner en la página que recibe los datos del formulario:

<%
if isnumeric(request.form("n1")) and isnumeric(request.Form("n2"))then
   response.write(cint(request.Form("n1"))+cint(request.Form("n2")))
else
   response.write("No has introducido datos numericos")
end if
%>

En este caso comprobaríamos que los datos del formulario son numéricos.

Si se envia con el Method=Get se recoge la informacion con la funcion Request.Querystring(“nombre”).

Get envia los datos a traves de la URL (limita a 10 los datos que se pueden enviar)

Para validar datos de un formulario debo comprobar los datos antes de operarlos.

• Isnumeric(valor) si valor es numerico devuelve True si no False
• Isdate(valor)
• Isempty(valor) si esta vacio true
• Isnull(valor) si no hay ningun dato
