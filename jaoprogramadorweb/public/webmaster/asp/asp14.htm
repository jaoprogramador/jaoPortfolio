Mostrar Registros en una tabla dinámica

Cuando queremos mostrar unos registros de una tabla de una base de datos, y mostrarlos en una página, hay que en primer lugar hacer una conexión a la base de datos, como explicamos en el apartado anterior, luego hacer una consulta SQL y finalmente hacer una tabla dinámica que muestre los resultados de la consulta.

Ejemplo

<%
set con=server.CreateObject("ADODB.CONNECTION")
con.open("coches")'nombre de la DNS a la BD
set rec=con.execute("select * from users where login like 'j%'")
response.Write("<table border=1><tr><td>Usuario<td>Clave")
while not rec.EOF
response.Write("<tr>")
response.Write("<td>"&rec("login")&"<td>"&rec("passw"))
rec.movenext()
wend
rec.close()
con.close()
set con=nothing
set rec=nothing
%>
