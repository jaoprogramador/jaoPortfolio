El evento onClick

Sin lugar a dudas, este es uno de los eventos mas conocidos por javascript y se produce cuando el usuario efectua un click con el boton izquierdo del ratón sobre algún elemento de un formulario o un enlace.

La sintaxis básica de este evento es la siguiente:

<input type="button" value="pulsa" onClick="function ">

El tipo de elemento puede ser un boton , una caja de comprobación o un boton de reset por ejemplo, mientras que la función es la instrucción o instrucciones a procesar cuando se produzca el evento.
