El evento onmove y onresize

El evento onmove se genera cuando el usuario mueve una ventana o frame de la pantalla principal del navegador. Cuando un usuario modifica el tamaño de la ventana o del frame actual, se genera un evento onresize. Estos eventos están incorporados en los objetos window y frame.

<HTML>

< BODY onresize="alert('La ventana esta siendo redimensionada')" onmove="alert('la ventana se esta moviendo')">

</BODY>

</HTML>
