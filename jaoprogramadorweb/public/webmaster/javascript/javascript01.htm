Primeros pasos en Javascript

Debemos de remarcar que Javascript diferencia entre mayúsculas y minúsculas. Como en todos los lenguajes javascipt permite la introducción de comentarios de una sola línea precedidos de // y de varias líneas que irán entre /* */.

Los corchetes se utilizan para definir fragmentos de código de manera que estos no se junten con el resto del programa.

Por ejemplo, todo el código que ejecuta en una función debe ir bien diferenciado del resto mediante corchetes:

<HTML>

<BODY>

<SCRIPT LANGUAJE="Javascript">

//definicion de la funtion

for(cuenta=0;cuenta<4;cuenta++)

//inicio del codigo de la función

{

alert("ejemplo de ventana de alerta JAO");

}

</SCRIPT>

</BODY>

</HTML>

 

La misión del punto y como en javascript es la de separar distintas sentencias que se encuentran en la misma línea. También puede indicar el final de una sentencia que ocupa varias líneas.
