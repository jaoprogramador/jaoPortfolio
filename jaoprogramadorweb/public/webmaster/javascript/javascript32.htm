onkeydown, onkeypress y onkeyup

Como el lector habrá podido imaginar, estos eventos se generan cuando el usuario efectúa alguna acción con las teclas.

•  onkeydown. Se ejecuta cuando se pulsa una tecla

•  onkeyup. Se ejecuta cuando se deja de pulsar una tecla.

•  onkeypress. Se ejecuta cuando se mantiene pulsada una tecla

Un ejemplo de uno de estos eventos sería:

<SCRIPT LANGUAJE="Javascript">

function avisoPulsacion(){

alert('Has pulsado una tecla');

</SCRIPT>

< BODY >

<FORM name="datos">

</BODY>
