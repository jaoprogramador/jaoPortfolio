El evento onReset

El evento onReset se produce cuando se pulsa un boton de reset en un formulario. Su sintaxis es la siguiente: <input type="reset" onReset="funcion"> Un ejemplo sería el siguiente:

<HTML>

<SCRIPT LANGUAJE="Javascript">

function avisoReset(){

if(confirm("los datos del formulario se van a borrar")){

document.datos.reset();

}

</SCRIPT>

< BODY >

<FORM name="datos">

<textarea rows=3 cols=40 onSelect="alert('Se ha seleccionado texto en el formulario')">Este es el contenido de un textarea JAO</textarea>

<input type="reset" onReset=" avisoReset()">

</FORM>

</BODY>

</HTML>
