Argumentos de las Funciones

Javascript posee una serie de objetos predefinidos, pero tambien podemos crear nuestros propios objetos según sea necesario. Los pasos para crear un nuevo objeto son los siguientes:

•  Definir un tipo de objeto a través de una función

•  Crear una instancia del objeto usando la palabra new

Así para definir un tipo de objeto deberemos de especificar el nombre de la fución , sus propiedades y metodos.

 

Un ejemplo podría ser una función llamada ordenador con tres propiedades marca, cpu y memoria:

Function ordenador (marca,cpu, memoria){

this.marca=marca;

this. cpu = cpu;

this. memoria = memoria;

}

A continuación crearemos un objeto ordenador al que asignaremos valores específicos a sus propiedades:

miOrdenador=new ordenador("HP","Pentium150","64Mb");

Ahora podríamos mostrar en pantalla las propiedades de los objetos definidos.
