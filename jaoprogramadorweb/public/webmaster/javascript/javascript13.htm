El objeto Array

Una técnica importante de programación es el uso de arrays o matrices. Podriamos definir un array o una matriz como un conjunto de datos de un mismo tipo identificados por un indice. Para comprender mejor este objeto iremos desarrollando una matriz de ejemplo. En primer lugar, deberemos definir la matriz, para ello utilizaremos el objeto Array().

 

A continuación, se deberá ir incluyendo los elementos de esa matriz. Si recordamos fundamentos de programación básica, veremos que cada elemento de una matriz está identificado mediante un número al que llamaremos indice.

 

OpenAustralia= new Array();

OpenAustralia[0]="Sergi Brugera";

OpenAustralia[1]="Alex Corretja";

OpenAustralia[2]="Meter Sampras";

 

A simple vista podemos ver se han introducido tres elementos en la matriz pero para matrices mayores hay propiedades del objeto Array llamada length, que facilita el número de elementos disponibles en la matriz. Podemos distinguir los siguientes métodos:

- join(). Método encargado de agrupar todos los elementos en una cadena de caracteres , separados por comas.

- reverse(). Este método invierte el orden de colocación de los elementos. El primero pasa a ser el último y viceversa.

Sort(). Este método devuelve de manera ordenada los elementos de la matriz.

 

Una situación que puede darse cuando se trabaja con matrices es la de borrar algún elemento . Javascript no permite eliminarde manera directa elementos de una matriz. Sin embargo podemos dar una valor nullo al elemento. Según nuestro ejemplo , veamos como eliminar los elementos tres y cuatro de la matriz

OpenAustralia[2]="";
