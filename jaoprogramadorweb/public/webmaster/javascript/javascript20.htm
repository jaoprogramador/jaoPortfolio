el objeto location

La propiedad location del objeto window contiene información sobre el URL completo de un documento actual a diferencia de la propiedad location del objeto document que se encarga de cargar un nuevo documento. Su sintaxis es la siguientes: window.location.propiedad

Veamos a continuación alguna de las propiedades del objeto location:

•  protocol especifica el inicio de la direccion

•  hash especifica el nombre del enlace en la url

•  host determina el nombre del ordenador y del puerto

•  href especifica la dirección completa

•  port especifica el puerto de comunicaciones
