El evento onSelect

Este evento hace que se ejecute un script cuando se seleccione texto dentro de un elemento de un formulario ya sea una casilla de texto o de area de texto. La sintaxis es similar a las vistas anteriormente.

<input type="tipo de elemento" onSelect="funcion" >

Veamos un ejemplo:

 

<HTML>

<SCRIPT LANGUAJE="Javascript">

function verificarDato(){

alert('ha cargado la ventanaJAO');

</SCRIPT>

< BODY >

<FORM >

<textarea rows=3 cols=40 onSelect="alert('Se ha seleccionado texto en el formulario')">Este es el contenido de un textarea JAO</textarea>

</FORM>

</BODY>

</HTML>
