Core/each: each del core de jQuery

Hay algo que tenemos que tener en cuenta, en la medida de lo posible, cuando creamos
código Javascript: crear un código de calidad y lo más corto posible. Para ello también nos
facilitan mucho las cosas los frameworks y métodos como each(), que veremos en este
artículo.
El método each() pertenece al juego de funciones del CORE de jQuery, cuyas particularidades
ya comenzamos a analizar en el manual de jQuery. Se trata de un método para realizar
acciones con todos los elementos que concuerdan con una selección realizada con la función
jQuery -también llamada función $()-. Útil porque nos da una manera cómoda de iterar con
elementos de la página y hacer cosas con ellos más o menos complejas de una manera rápida
y sin utilizar mucho código para definir el bucle.
