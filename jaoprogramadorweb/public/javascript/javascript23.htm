Los eventos en javascript

Se llama evento a un suceso que ocurre en el sistema cuando un usuario ejecuta algún tipo de acción. Un evento debe asociarse a un elemento HTNL, no a un código javascript.

En este capítulo serán tratados los manejadores de eventos disponibles en javascript 1.2.

Veamos a continuación, como introducción un script que utiliza el evento onClick:

 

<HTML>

<BODY >

<input type=”button” value=”pulsa” onClick=”alert(‘has pulsado el boton JAO') “>

</BODY>

</HTML>
