Estructuras condicionales

Al igual que sucede con la mayoría de los lenguajes de programación en determinados momentos es necesario controlar el desarrollo del programa para que este tome decisiones por nosotros. Javascript dispone de los siguientes comandos:

•  If: se emplea cuando es necesario ejecutar un fragmento de código en función de los valores de determinadas variables. La estructura if tiene la siguiente sintaxis:

If(condicion){

//líneas de código de programa si se cumple la condición

}

•  For, nos permite hacer que un proceso se repita determinadas veces. Para ello se tienen que establecer los siguientes valores:valor inicial, condición, la ejecución de las intrucciones y la actualización . Veamos la estructurabásica de un bucle for:

for(valor inicial; condicion<11; contador++)

•  While. Ejecuta de manera constante una determinada secuencia de código siempre que se cumpla una condición. Veamos la estructura básica de un bucle while:

while(condicion)

}

•  do while, este bucle no estaba implementado en las primeras versiones de javascript. La misión de este bucle es la de colocar una expresión condicional después de las expresiones que deben ejecutarse, de esta manera seguro que las instrucciones son ejecutadas por lo menos una vez. Su estructura es la siguiente:

do{

instrucciones

} while (condicion);
