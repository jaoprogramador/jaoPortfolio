Funciones

Las funciones es uno de los pilares en los que se basa JavaScript. Podríamos definir una función un conjunto de sentencias javascript que realizan una tarea especifica. Las partes que definen una función son:

•  El nombre de la función

•  La lista de argumentos de la función, encerrados entre paréntesis y separados por comas(“,”).

•  Las sentencias en javascript que definen la función son, encerradas entre llaves(“{}”);

La sintaxis de definición de una función sería:

Function nombreFuncion(parametro1, parametro2){

instrucciones

}
