Introducción

Javascript es un lenguaje de programación creado por Netscape con el objetivo de integrarse en html y facilitar la creación de paginas interactivas sin necesidad de utilizar scripts de CGI o Java.

El código de un programa de javascript, llamado script se introduce directamente en el documento html y no necesita ser compilado, es el propio navegador el que se encarga de traducir dicho código.

Destacar también que no todos los navegadores soportan javascript. Vamos a ver a continuación las versiones soportadas por los navegadores mas populares:

Javascript 1.0 : Netscape Navigator 2.0 o superior , Microsoft Internet Explorer 3.0

Javascript 1.1 : Netscape Navigator 3.0 o superior , Microsoft Internet Explorer 4.0

Javascript 1.2 : Netscape Communicator 4 o superior , Microsoft Internet Explorer 4.0

Sabemos que un programa en Javascript no debe ser compilado, ya que el navegador se encarga de leerlo cuando se carga la página, pero esto no significa que todos los códigos se ejecuten nada mas cargarse el documento html, algunas funciones permanecen en estado latente hasta que el usuario realice una acción determinada, como podria ser presionar sobre un enlace.

El primer paso es diferenciar de un documento lo que es Javascript del resto. Para ello html dispone de una etiqueta que define el principio y el final de un código Javascript.

Para definir el inicio de un programa o código se debe utilizar la etiqueta <SCRIPT LANGUAJE=” Javascript” y determinar el final con </ SCRIPT >.

El atributo LANGUAJE indica al navegador el lenguaje script utilizado. En el caso que nos ocupa el nombre a poner debe ser Javascript. Si se omite la especificación del lenguaje el navegador interpreta que el lenguaje utilizado es Javascript.

Un ejemplo de código javascript sería el siguiente:

<HTML>

<BODY>

<SCRIPT LANGUAJE=”Javascript”>

document.write(“Primer ejemplo JAO!!!”);

</SCRIPT>

</BODY>

</HTML>
