El evento onAbort

El evento onAbort se produce cuando se bloquea la carga de una imagen, pulsando por ejemplo el boton STOP. Su sintaxis es la siguiente: <IMG SRC=”imagen” onAbort=funcion>

En el siguiente script de ejemplo, si el usuario pulsa el boton stop o accede a cualquier otro enlace, un mensaje de aviso le aparecerá en la pantalla.

 

<HTML>

< BODY >

<IMG SRC=”imagen.jpg” onAbort=alert(‘Falta por cargar la imagen')>

</BODY>

</HTML>
