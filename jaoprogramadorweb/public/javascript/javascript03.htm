Tipos de variables

Javascript puede manejar tres tipos de de datos distintos decidiendo por nosotros el tipo de variable que deberá emplear durante la ejecución del script. Los tres tipos de variables son:

1.- Variables de cadena, es aquella que contiene un texto o una cadena de caracteres: Ej: var nombre=”JAO”;

2.- Variables numéricas, es aquella que contiene un valor númerico. Ejemplo: var num=1;

3.- Variables booleanas, son aquellas que solo admiten dos valores trae o false

Un cuarto tipo podrían ser los datos los datos nulos. Los valores (null) se utilizan para comprobar si a una variable se le ha asignado un valor o no. Así pues, representa un valor nulo para cualquier tipo de variable, por el contrario, una variable que no ha sido inicializada tiene un valor undefined.
