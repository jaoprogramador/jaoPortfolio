El objeto Boolean

Este objeto se utiliza para convertir datos no booléanos en boléanos. Su sintaxis es la siguiente:

NuevoObjeto=new Boolean(valor)

Las reglas elementales para utilizar este tipo de objetos son las siguientes:

•  Cuando se crea un objeto sin proporcionar datos , el nuevo objeto toma el valor de false.

•  Cuando se facilita el valor true o cualquier otro texto entrecomillado, el resultado del nuevo objeto es siempre true.

•  El nuevo objeto contendrá false si se proporciona un valor 0, false sin comillas o una cadena de texto vacía.
