Primera página ASP

Cómo hacer una página ASP

La generación de las tablas dinámicas se hará con VBS, el código estático con HTML. El contenido se genera en el servidor (html+VBS+…), el trabajo se hace en el servidor, y se envia HTML al cliente. El resultado de la página Asp es el que ve el cliente en el navegador...

Así pues, tras haber introducido las etiquetas típicas de HTML com <html><body> cuando queramos empezar con el lenguaje VBS habrá que introducir <% ...líneas de código ... %>. Previamente habra que haber puesto antes de la etiqueta <html> esta linea de código <%@LANGUAGE="VBSCRIPT" CODEPAGE="1252"%> para indicar el lenguaje de la página y de codigos (el idioma en el que está cifrada).

EJEMPLO:

<%@LANGUAGE="VBSCRIPT" CODEPAGE="1252"%>
<html>
<head>
<title>Documento sin t&iacute;tulo</title>
<meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1">
</head>
<%

author = "Juan Arana Olalde"

department= "Desarrollo"

%>
Esta página fue actualizada <B>hoy</B>, por <%= author %> del

departamento de <%= department %>.
<body>

</body>
</html>

para comprobar el resultado solo tendremos que guardar la página con extensión .asp en el directorio del servidor local, que en este caso será: C:\Inetpub\wwwroot\NombreDelSitio\pagina.asp . Tras esto tendremos que abrir el explorador, Internet explorer por ejemplo, y escribir http://localhost/NombreDelSitio/index_marcos.htm.

NombreDelSitio es el nombre que tiene la carpeta creada en el directorio C:\Inetpub\wwwroot\ donde hemos puesto la página creada en asp. Es posible que si tenemos instalado un Firewall nos de problemas así que es recomendable cerrar esa aplicación antes de comprobar en el explorador los resultados de una página asp.
