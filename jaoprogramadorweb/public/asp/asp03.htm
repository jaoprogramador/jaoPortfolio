Directivas de proceso

Tipos:

Se ponen al principio del código de la página asp y se distinguen las siguientes:

- <%@ LANGUAGE= Vscript %>
indica lenguaje que se utiliza

- <%@ CODEPAGE %>
especifica la página de códigos que utiliza (japones, occidental…)

- <%@ ENABLESESSIONSTATE %>
establece una sesión, para que se mantengan activas las variables durante la sesión.

- <%@ TRANSACTION %>
indica que se esta ejecutando una operación en una transacción, para enviar un conjunto de documentos a la vez, no enviando los documentos por separado.

.- <%@ LCID%>
Identificador de Codigo Regional indica la zona en la que se esta (España 34….). Importante al utilizar fechas
