Condicionales en ASP

If - Else y Select Case en ASP

If condicion then comandos

Else comandos

End if


O con una condicion dentro de otra.

If condicion1 then comandos

Else if condicion2 then comandos

Else

End if


Con pocas instrucciones

If condicion then comando1 else comando2


Otra estructura condosional, para elegir una caso dependiendo del valor de una variable

Select Case variable

Case valor1
Comandos

Case valor2
Comandos

Case else
Comandos

End select

Ejemplos

Veremos como con una sentencia For podemos filtrar el valor de una variable mediante mensajes:

<%
Dim I
I = 8
If I = 8 Then
Response.Write (" I contiene el valor 8")
Else
Response.Write ("I contiene un valor distinto de 8")
End If
%>
