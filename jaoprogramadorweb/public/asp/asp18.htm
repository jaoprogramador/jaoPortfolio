Insertar registros en ASP

Insertar registros a una BD desde un formulario

ASP nos permite la inserción de registros en una base de datos. Para ello, necesitaremos dos páginas .asp:

- Una contendrá un formulario que pasará los campos que deseemos introducir los datos en la tabla de la base de datos.

- Otra que recibirá los parámetros y los introducirá los datos en la tabla de la base de datos a través de un instrucción SQL.

Ejemplo

1º.- La página del formulario contendrá un formulario con estas líneas dentro del BODY:

<form name="form1" method="post" action="datosEj29.asp">
<table width="75%" border="0">
<tr>
<td width="10%">Login:</td>
<td width="90%"><input name="log" type="text" id="log"></td>
</tr>
<tr>
<td>PassW:</td>
<td><input name="pas" type="text" id="pas"></td>
</tr>
<tr>
<td>&nbsp;</td>
<td><input type="submit" name="Submit" value="Enviar"></td>
</tr>
</table>
</form>

2º.- La página que recibirá los datos y los introducirá a la base de datos se llamará "datosEj29.asp" y contendrá las siguientes líneas dentro del BODY:

<%
set con=server.CreateObject("ADODB.CONNECTION")
con.open("coches")'nombre de la DNS a la BD
set rec=con.execute("insert into users (login, passw)values('"&request.form("log")&"','"&request.form("pas")&"')")
con.close()
set con=nothing
set rec=nothing
%>
El usuario <b><% response.write(request.Form("log"))%></b> se registro correctamente

Con esto, introduciremos a la tabla "users" en los campos "login" y "passw" los valores del formulario que son "log" y "pas".
