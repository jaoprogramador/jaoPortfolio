Variables

Variables en ASP

Las variables si no se declaran se generan del tipo variant, pero ocupa mucha memoria (4 byte). Además conviene declararlas para reconocerlas dentro del código. Se declaran dentro de la parte de VBS con el comando dim.
El nombre de la variable debe empezar por letra, no puede sobrepasar de 255 caracteres, no puede contener signos de puntuación. Para reconocer una variable de varias palabras es poner la primera letra de cada palabra en mayuscula: (NumeroDePuntos).

Para que la declaracion de variables sea obligatoria escribir al inicio del fichero <%Option explicit%>

Ejemplos

<%
dim x,y
x=8
response.write("Julen Guerrero tiene el n "& x&"<br> a la espalda" )

%>

Las variables dentro de la hoja pueden ser públicas (Public: valida solo en el trozo de código en el que este) o privadas (Private: valida en toda la pagina)

<%
public a
a= 12
response.Write("El valor de a es: <b>"&a&"</b><br>")
%>
<%
Response.Write("El valor de a es publico a =<b>"&a&"</b>")
%>

La privacidad de las variables tiene sentido en el caso de definir funciones que veremos en el siguiente apartado.
