Control Data

Práctica 44.- Aplicación que nos permite realizar todas las operaciones posibles sobre una base de datos que lleva el control de empleados de una empresa, así como movernos por los distintos registros. Podremos ver los distintos RG que contiene la base de datos en las distintas cajas de texto.

 

Control ADO

Práctica 44B.- Aplicación que nos permite realizar todas las operaciones posibles sobre una base de datos que lleva el control de empleados de una empresa, así como movernos por los distintos registros. Podremos ver los distintos RG que contiene la base de datos en las distintas cajas de texto.

 

Control ADO y ficheros

Práctica 45.-Aplicación que nos permite visionar los distintos registros de una base de datos y crear un fichero con los datos de ese registro, al cliquear sobre una lista.

  

Control ADO y DB Grid

Práctica 46.- Aplicación que nos muestra una tabla de una base de datos por medio de un control DBGrid a través de una consulta SQL.

  

Control ADO, DataGrid y ActiveX

44AdoActiveX.- Aplicación que nos permite introducir, editar, borrar, buscar un registro en una base de datos estudiantes para posteriormente visualizarlo, si lo deseamos en un DataGrid de otro formulario.

  

Control ADO, DataGrid, ActiveX, DataList y DataCombo

DB_Ado_DataCombo.- Aplicación que nos permite ver los campos Titulo y ISBN de la base de datos Biblio junto con los distintos nombres en un DataCombo. También nos permite desplazarnos a otro formulario donde podemos ver en dos DataGrid los autores y los libros. En este formulario también aparecen un DataCombo que modificará en contenido de un datalist en función del Identificador que contenga ese autor.
