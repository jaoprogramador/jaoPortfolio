Requisitos de un proyecto EJB 3.0

Para desarrollar un contenedor de EJBs 3.0 que interactúe con una BD y un cliente se necesitan entre otras cosas:

- Tener instalada la maquina virtual de java.

- Tener un IDE con las actualizaciones de EJB 3.0, en nuestro caso usaremos NetBeans, pero podríamos haber usado Eclipse, JBuilder...

- Tendremos que incluir las siguientes librerías en el contenedor de EJBs: ejb3-persistence.jar, jboss-aop-client.jar, jboss-aspect-jdk50-client.jar, jboss-ejb3-common-client.jar y jboss-ejb3-core-client.jar.

- Un servidor con soporte para EJB 3.0 que contendrá los EJBs, en nuestro caso JBoss 5.1 y un servidor Tomcat 5.5 que contendrá las JSPs que interactuarán con los EJBs.

- Tendremos que montar un contenedor de EJBs y un cliente para llamar a los EJBs que interactuarán con la base de datos y las JSPs o JSFs.

- Una base de datos que en nuestro caso será MySQL.
