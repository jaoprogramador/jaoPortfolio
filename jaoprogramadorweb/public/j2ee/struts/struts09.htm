Configurar Servidor:

Bien, lo primero que haremos sera descargarnos en IDE en mi caso NetBeans 6.7.1. aunque podríamos haber utilizado JBuilder, Eclipse o cualquier otro. Lo siguiente será descargarse el servidor Tomcat versión 5.5 para integrarlo con nuestro proyecto web.

Tras esto, ejecutamos el NetBeans y en el marco de la izquierda seleccionamos la pestaña "Prestaciones" y pinchamos boton derecho del ratón sobre la opciónn servidores y seleccionamos "Agregar servidor". Nos aparecerán una lista de servidores y nosotros seleccionaremos Tomcat 5.5. y le damos al botón de "Siguiente" y nos aparecerá una ventana donde especificar donde tenemos almacenado nuestro servidor Tomcat( Server Location). Pues bien, pulsamos el botón "Browse" y seleccionamos el directorio donde descargamos el Tomcat.

Posteriormente checkeamos "Use Private Configuration Folder (Catalina Base)" y seleccionamos el directorio donde tenemos instalado el Tomcat.

Luego, introducimos en "Username:" manager, y en "Password:" admin y pulsamos el botón "Terminar".

Es entonces cuando podremos ver que se nos ha agregado el servidor Tomcat en el marco de la izquierda en la pestaña "Prestaciones" .Pues bien, pinchamos boton derecho del ratón sobre "Tomcar 5.5" y seleccionamos "Start" para arrancar el servidor. Crear Proyecto Web con Struts:

Lo primero, en el marco de la izquierda de Netbeans seleccionamos la pestaña "Proyecto" y pinchamos boton derecho del ratón dentro del marco y seleccionamos "Proyecto Nuevo" en "Categoría": Java Web, y en "Proyectos": Web Application, pulsamos botón "Siguiente".

Nos aparece una ventana para introducir el nombre del proyecto, en mi caso JAO y seleccionamos el directorio donde queremos crear nuestro proyecto. Pulsamos el botón "Siguiente".

Ahora nos sale una ventana donde pondremos el servidor, en nuestro caso, Tomcat 5.5, que es el servidor que configuramos anteriormente. Nos aparece la versión de Java que tenemos instalada y el Context Path introducido anteriormente(/JAO) . Le damos a "Siguiente" y es entonces donde le especificamos el Framework que tendrá nuestra aplicación web: Struts 1.3.8, asi que lo chequeamos y pulsamos "Terminar". Esto nos crea el proyecto JAO con las librerias necesarias para trabajar con Struts, el struts-config.xml donde definiremos la navegación y la correspondencia ActionForm-FormBean, un par de JSPs, el fichero de recursos, el validation.xml para realizar validaciones de los formularios...

Y ya nos podemos poner a programar...
