Técnicas para corregir la forma

Una biblioteca de etiquetas como el que viene con el servidor Blazix, pueden no estar disponibles en su entorno. ¿Cómo se puede permitir características similares sin necesidad de utilizar una biblioteca de etiquetas?

Es un poco tedioso, pero se puede hacer. Básicamente, debe modificar cada etiqueta de código HTML, y poner en un valor predeterminado. Los siguientes ejemplos muestran cómo modificar GetName.jsp para ofrecer características similares a BLX: getProperty pero con manual edición de etiquetas HTML:

<jsp:useBean id="user" class="user.UserData" scope="session"/>
<HTML>
<BODY>
<FORM METHOD=POST ACTION="SaveName.jsp">
¿Cuál es tu nombre? <INPUT TYPE=TEXT NAME=username
SIZE=20 VALUE="<%= user.getUsername() %>"><BR>
¿Cuál es tu dirección de e-mail? <INPUT TYPE=TEXT
NAME=email SIZE=20
VALUE="<%= user.getEmail() %>"><BR>
¿Cuál es tu edad? <INPUT TYPE=TEXT NAME=age
SIZE=4 VALUE=<%= user.getAge() %>
<P><INPUT TYPE=SUBMIT> <P>
</FORM>
</BODY>
</HTML>

Como puede ver, esto simplemente implica la adición de un "valor" de campo en las etiquetas INPUT, e inicializar el campo con una expresión!

Para controlar las excepciones durante el proceso de entrada, un enfoque sencillo es utilizar "Cadena" campos de la soja, y hacer la conversión al tipo de dato objetivo usted mismo. Esto le permitirá manejar excepciones.
