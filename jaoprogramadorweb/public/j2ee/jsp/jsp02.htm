Su primera JSP

JSP simplemente pone Java dentro de páginas HTML. Se puede tomar cualquier página HTML existente para cambiar su extensión a ". JSP" en lugar de ". Html". De hecho, este es el ejercicio ideal para su primera JSP. Tome el archivo HTML que utiliza en el ejercicio anterior. Cambie su extensión de ". Html" a ". Jsp". Ahora carga el archivo nuevo, con el jsp "." Extensión, en el navegador.

Verá la misma salida, pero llevará más tiempo! Pero sólo la primera vez. Si usted vuelve a cargar de nuevo, se carga normalmente.

Lo que está ocurriendo detrás de las escenas que su JSP se está convirtiendo en un archivo Java, compilado y cargado. Esta compilación sólo ocurre una vez, así que después de la primera carga, el archivo no se tarda mucho en cargar más. (Pero cada vez que cambies el archivo JSP, será re-compilado de nuevo.)

Por supuesto, no es muy útil para escribir sólo las páginas HTML con la extensión. Jsp! Procedemos ahora a ver lo que hace JSP tan útil.
