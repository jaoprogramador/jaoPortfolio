Mezcla de Scriptlets con HTML

Ya hemos visto cómo utilizar la "salida" variable para generar la salida HTML desde un scriptlet. Para más complicado HTML, utilizando la variable a cabo todo el tiempo pierde algunas de las ventajas de la programación JSP. Es más sencillo scriptlets mezcla y HTML .

Supongamos que usted tiene para generar una tabla en HTML. Esta es una operación común, y es posible que desee generar una tabla de una tabla de SQL o de las líneas de un archivo. Pero para mantener nuestro sencillo ejemplo, vamos a generar una tabla que contiene los nÃºmeros de 1 a N. No es muy útil, pero le mostraría la técnica.

Aquí está el fragmento JSP para hacerlo:

<TABLE BORDER=2>
<%
for ( int i = 0; i < n; i++ ) {
%>
<TR> <TR>
<TD> Número </ TD>
<TD><%= i+1 %></TD>
</TR>
<%
} )
%>
</TABLE>

Usted tendrá que facilitar una variable int "n" para que funcione, y entonces sería una simple tabla con "n" filas.

Lo importante a destacar es cómo el%> y <% personajes aparecen en el centro del bucle "for", que le permiten colocar de nuevo en HTML y luego regresar a la scriptlet.

Los conceptos son fáciles aquí como se puede ver, usted puede abandonar el scriptlets, escribir HTML normal, y volver a la scriptlet. Las expresiones de control, como un "tiempo" o un bucle "for" o "si "expresión controlaría el HTML también. Si el código HTML se encuentra dentro de un bucle, que sería emitida una vez por cada iteración del bucle.
