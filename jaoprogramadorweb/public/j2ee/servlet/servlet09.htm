Proporcionar Información de un Servlet

JavaSoft suministra tres componentes JDBC como parte del JDKAlgunas aplicaciones, como La Herramienta de Administración del Java Web Server, obtienen información sobre el servlet y la muestran. La descripción del servlet es un string que puede describir el proposito del servlet, su autor, su número de versión, o aquello que el autor del servlet considere importante. El método que devuelve esta información es getServletInfo, que por defecto devuelve null. No es necesario sobreescribir este método, pero las aplicaciones no pueden suministrar descripción de nuestro servlet a menos que nosotros lo hagamos. El siguiente ejemplo muestra la descripción de BookStoreServlet.

public class BookStoreServlet extends HttpServlet {
...
public String getServletInfo() {
return "The BookStore servlet returns the " +
"main web page for Duke's Bookstore.";
}
}
