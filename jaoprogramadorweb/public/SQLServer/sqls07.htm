Declaración de Variables

1.- Ejemplo de como declarar una variable de tipo entera.

DECLARE @TOTAL INT

  

Asignación de variables

2 .- Ejemplo de como asignar valores a una variable de tipo entera.

use libreria
declare @total int
select @total=12

  

Asignación mediante SELECT

3 .- Ejemplos de como asignarle un valor a una variable a través de una consulta SELECT.

use libreria
declare @total int
select @total=12
select @total=count(titulo) from libros

  

Impresión de variables por pantalla

3 .- Ejemplos de como recoger el resultado de una consulta en una variable y devolverlo por pantalla.

use libreria
declare @total int
select @total=count(titulo) from libros
print @total

  

Manejo de variables con operadores y funciones

3 .- Ejemplos de como asignarle un valor a una variable a través de una consulta SELECT modificando su resultado con funciones como COUNt o AVG.

use libreria
declare @total int
select @total=count(titulo) from libros
print @total
