Introducción

PHP ("PHP: Hypertext Preprocessor") es un lenguaje de programación de alto nivel que se ejecuta en el servidor devolviendo páginas HTML de las que, además, puede formar parte. Dicho de otra forma: Permite crear páginas web dinámicas. (Para ello basta tener cargado el módulo PHP en el servidor web apache)

Es open source o sea gratuito, está disponible para los principales sistemas operativos, permite trabajar con diferentes bases de datos y soporta diferentes protocolos.

Además es un potente lenguaje de programación capaz de utilizar, por ejemplo, objetos y clases. Para utilizarlo como tal e independientemente de la web, crearemos los programas con un editor y los ejecutaremos tecleando: php programa.php

Vamos a utilizar php fundamentalmente para trabajar con mySQL desde la web, esto significa que además del módulo de php necesitaremos tener también instalado el servidor apache.

Si hemos hecho la instalación de servidor de suse ambos estarán instalados. Si esto no fuera así los instalaríamos desde yast o mediante RPM tras bajarnoslo de la web. El sitio oficial es http://www.php.net/ Este site cuenta con varios enlaces: download, documentación, ... También hay una zona en castellano http://www.php.net/manual/es/ en la que podemos encontrar manuales, notas de usuarios...
