Arrays

- Los arrays se utilizan dentro de un programa cuando queremos tener varias variables del mismo tipo relacionados entre sí.

- Al declarar un array se consigue que el sistema reserve en memoria un conjunto de posiciones contiguas en las que introduciremos los datos.

- Para declarar un array en C: Tipo_de_datos nombre_array[nº_posiciones];Ejemplo: 

int nombres[12]; ---> Reservamos en memoria un espacio de 12 posiciones
                            en las que se almacenarán caracteres para hacer
                            referencia al array "nombres"
