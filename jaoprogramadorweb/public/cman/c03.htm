Estructura de un programa en C

Un programa C se forma por funciones que tiene las siguientes características.

- Pueden ser recursivas - No se pueden anidar definiciones de función - Las funciones que componen un programan no tiene porque estar en el mismo fichero fuente - Las variables del programa pueden ser:
a) Internas a una función: Locales b) Externas a una función: Conocidas como en el fichero en el que están o conocidas para todo el programa.

El programa fuente en C hay que compilarlo para obtener el programa objeto. Este proceso tiene dos pasos:

Preproceso: se encarga de la sustitución de macros, la inclusión de otros ficheros y la compilación condicional.

#include<stdio.h>
		main()
		{
		   printf("\nEste es el primer programa");
		}

# Para que el procesador incluya en el programa el contenido del fichero <stdio.h>

Función main() : Función principal de un programa C. Todo programa tiene un función main() a partir de la cual va a empezar la ejecución de nuestro programa.
