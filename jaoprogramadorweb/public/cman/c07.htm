Funciones estándar en C

a) Función printf().

- Se utiliza para producir salidas a pantalla - Los argumentos que se le pasan a la función printf()son: Un string de control que va entre " ", y en el se incluyen:
- Cadenas de caracteres - Especificadores de conversión
%C: Escribe el contenido de una expresión como carácter. %d: Escribe el contenido de una expresión como entero decimal. . %e: Escribe el contenido de una expresión como un número real.en notación científica. %f: Escribe una variable como un número real. %g: Escribe en forma e o f, el mas corto. % s: Escribe un string
- Caracteres especiales:
\n: imprimer un retorno de carro. \t: Imprimer un tabulador.
- Otros argumentos: expresiones que se emparejan con los especificadores de conversión respectivos: Ej: printf("\n %d",a); Ej: printf("%C %3c%5c",'A','B','C');

b) Función scanf().

- Se utiliza para la entrada de datos por teclado (normalmente) - Los argumentos que se le pasan a una función scanf() son: un string de control que contiene especificadores de conversión en el que se indica como deben interpretarse los caracteres que se introduzcan:
%c: recibe variable de tipo char %d: de tipo int(entero) %f: de tipo float(real) %df de tipo double %Lf: de tipo long double %s: de tipo string ** Direcciones de memoria: &--> operador de dirección.Ej: scanf("%d",&x);
