Declaración de constantes

Para declarar un valor constante en C, tenemos tres procedimientos:

1.- Utilizando el modificador de tipo const lo cual supone no poder variar el valor de la variable indicada a continuación: const tipo int A=7. 2.- Utilizando constantes simbólicas, se declaran mediante la directiva del procesador # define nombre_valor: #define PI 3.141521

Ventajas de utilizar constantes simbólicas: 1.- Se crean programas más legibles. 2.- Es más cómodo y fácil sustituir el valor de una constante en un fichero.
