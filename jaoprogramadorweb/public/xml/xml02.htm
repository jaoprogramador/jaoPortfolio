¿Qué es SGML?

SGML (Structured Generalizad Markup Language) es el predecessor del XML; éste es un metalenguaje que cumple con el standard ISO 8879, permite definir distintos tipos de documentos y cuyos objetivos son:

•  Proceso de documentos por ordenador

•  Separación de estructuras, contenido, y presentación

•  Independencia de sistemas y vendedores

Es decir, tenía las mismas características que el XML y se utilizaba, pero tenía varios inconvenientes:

•  complejidad

•  muchas opciones para aplicaciones especiales

•  sintaxis no legible

•  SGML diseña herramientas demasiado complicadas

Por tanto si al SGML le añadimos Internet(antes no estaba preparado para la presentación de documentos de internet) y le eliminamos su complejidad y otras características que le hacen legible, obtenemos el XML.
