Introducción a Bases de Datos con ASP

ASP nos permite conectarnos a sistemas gestores de bases de datos, como Oracle, SQL Server para interactuar entre las páginas y la base de datos pudiendo hacer todas las operaciones que nos dejan hacer las bases de datos: inserción, eliminación, actualizado y consulta de registros.

Como nosostros una introducción trabajaremos con bases de datos sencillas de Microsoft Access, aunque si a nuestras páginas accediesen numerosos usuarios que trabajasen simultaneamente contra la base de datos, deberiamos de tener un sistema gestor de base de datos mas potente como pudiesen ser Oracle u Microsoft SQL Server.

Para conectarse con las bases de datos y añadir y extraer datos se utilizan los objetos ADO. Debe estar ODBC instalado en el sistema Lo primero es conectarse a la basa de datos: objeto connection
Set con = server.createobject(ADODB.CONNECTION)
con es la referencia al objeto de conexión con la base de datos
metodos del objeto conexión
Con.open(“cadena DSN”)

Con.open(“Driver ={Microsoft Access Driver (*.mdb)}; DBQ=”&server.mappath(“db/NombreBaseDatos.mdb”))
Para asociar una base de datos utilizando herramientas administrativas del panel de control.
Con.close() para cerrar la conexión con la base de datos
Al final liberamos los objetos

Set con=nothing
