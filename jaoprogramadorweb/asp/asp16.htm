Propiedades y metodos del objeto Connection

• Con.ConnectionTimeOut= tiempo en el que corta la conexión con la base de datos si no se conecta
• Con.ConnectionString=”cadenaDSN” + con.Open() es lo mismo que se ha hecho antes para conectarse a la base de datos
• Con.State() devuelve true/false indicando si existe conexión con la base de datos

• Con.Open()
• Con.Close()
• Con.BeginTrans() inicio transaccion
• Con.Committrans() final de transaccion y envio de los datos a la base de datos

Transaccion: operación de acceso a la base de datos que se ejecuta de manera completa y sin conflictos

• Con.RollBackTrans() deshace la ultima transaccion
• Con.Execute(“instrucción SQL(que no sea sacar datos por pantalla)”)

Una instrucción SQL que no extrae datos devuelve nothing, si devuelve datos hay que guardarlos en un objeto de tipo recordset
