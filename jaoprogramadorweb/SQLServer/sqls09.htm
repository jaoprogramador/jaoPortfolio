While

1.- Ejemplos de utilizar una variable que funcione como un contador incrementable de 10 en 10 mediante la sentencia WHILE.

use libreria
declare @contador smallint
select @contador=1
while @contador!>10
begin
print @contador
select @contador=@contador+1
end

2 .- Ejemplos de como con la función POWER para mostrar los valores que tendría el número 2 elevado a distintos exponentes, desde el 0 hasta el 8 con la sentencia WHILE.

use libreria
declare @exponente smallint
set @exponente=0
while @exponente !>8
begin
print '2 elevado a '+ cast(@exponente as char)+'='+cast(power(2,@exponente)as char)
set @exponente=@exponente+1
end
