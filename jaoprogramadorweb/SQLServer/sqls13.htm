Procedimientos almacenados

1.- Ejemplos de como crear un procedimeinto almacenado que nos permite introducir registros a la tabla usuarios.

CREATE PROCEDURE [dbo].[ALTAUSUARIOS]
@nombre char(20),
@dni char(12),
@direccion char(30),
@tf char(10)

AS
insert into USUARIOS(nombre,dni,direccion,tf)
values(@nombre ,@dni,@direccion ,@tf)
GO

  

Procedimientos almacenados

2 .- Ejemplos de como crear un procedmiento almacenado que nos deja eliminar un usuario junto con sus datos a través del dni.

CREATE PROCEDURE [dbo].[BAJAUSUARIOS]
@dni char (12)
AS
delete from usuarios where dni=@dni;

GO
