Intoducción

Una función es un procedimiento que cuando se ejecuta devuelve un resultado. La sintaxis correspondiente a un a función es la siguiente:

      [Private|Public] [Static] Function nombre [(parametros)] [As tipo]

             [sentencias]

             [nombre=expresión]

             Exit Function

             [sentencias]

             [nombre expresión]

       End Function

Por compatibilidad con otros lenguajes, es aconsejable escribir los paréstesis cada vez que se llama a la función aunque ésta no tenga argumentos. Una llamada a la función puede formar parte de una expresión.

Sentencia For Each ... Next

La sentencia For Each ... Next se similar a la sentencia For, con la diferencia de que ahora se repite un grupo de sentencias por cada elemento de una colección de objetos o de una matriz (excepto matrices de elementos de un tipo definido por el usuario). Esto es especialmente útil cuando no conocemos cúantos elementos hay en la colección o en la matriz. Su sintaxis es la siguiente.

    For each elemento ln grupo

         [sentencias]
         [Exit For]
         [sentencias]
    Next elemento

Un ejemplo práctico pordía ser:

    Public Sub Sum(matrizx() as integer)
      Dim x , suma
      For each x ln matrizx
         suma = suma + x
      Next x
      msgbox suma
    End Sub
