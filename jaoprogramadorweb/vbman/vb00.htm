Introducción

Visual Basic es una herramienta para crear aplicaciones para Windows, que como su propio nombre indica, se realizan de una forma Visual, ya que nos permite crear los distintos controles sobre el formulario desde la barra de herramientas sin la necesidad de escribir una sola línea de código.

La palabra "Basic" hace referencia al lenguaje BASIC (Beginners All-Purpose Symbolic Instruction Code), que es uno de los lenguajes de programación.

Visual basic nos permite interaccionar con otras herramientas como pueden ser Microsoft Acces, Microsoft Excel, Microsoft Fox Pro, Microsoft SQL Server ... de manera que de programas sencillos podamos confeccionar aplicaciones mas complejas y elaboradas como podremos ver en ejercicios con bases de datos, ficheros...
