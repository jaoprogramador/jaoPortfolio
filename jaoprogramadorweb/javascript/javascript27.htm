El evento onChange

El evento onChange se produce cuando se modifica el valor de un elemento select, text o textarea en un formulario HTML.

Para que el evento funcione el usuario tiene que cambiar de elemento en el formulario, es decir, si estamos introduciendo en una casilla de texto, el evento onChange no se activará hasta que no pasemos a otra casilla. La sintaxis es la siguiente:

<input type=”tipo elemento” onChange=”funcion”>

Esta es sin duda una interesante opcion para validar la entrada de datos en formularios.

<HTML>

< BODY >

<FORM>

<input type=”text” value=” JAO” onChange =”alert(‘has cambiado el texto JAO') “>

</FORM>

</BODY>

</HTML>
