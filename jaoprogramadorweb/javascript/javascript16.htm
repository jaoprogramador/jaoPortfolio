Las funciones parseInt y parseFloat

Estas dos funciones devuelven un valor numérico a partir de una cadena. La función parseFloat() convierte una cadena en un número real en una coma flotante. Sila función encuentra un símbolo diferentes los signos + y -, los dígitos del 0 al 9 o el punto o el exponente, se devuelve el valo obtenido hasta ahí ignorando el resto de los caracteres .

Si el primer carácter no puede ser convertido se devuelve el valor NaN.

La función parseInt(cadena,base) convierte una cadena a una base numérica especificada. Por ejemplo: si la base es 2, la cadena puede ser un número binario en la forma “ 100101011” .

 

Si se encuentra un carácter que no es válido en la base especificada, se devuelve el valor obtenido hasta el momento. Si el primer carácter encontrado no es valido se devuelve el valor NaN.
